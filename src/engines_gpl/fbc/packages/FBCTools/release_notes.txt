RTC Tools 1.4 Release Notes
===========================

The major new features in this release are:

* Scenario trees for efficient stochastic optimization;

* Chance constraints for stochastic optimization;

* A set of fully SOBEK2-compatible components for classical feedback control;

* Linux support;

* CMake build system;

* BMI interface for communication with Deltares' Next Generation Hydro Software (NGHS).

For the fine details, please consult the Subversion log.
