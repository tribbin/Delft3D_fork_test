\section{Introduction}
This Developer's Guide concerns the hydrodynamic module, \DFLOWFM{}, of the Delft3D Flexible Mesh software
suite. This guide is intended for people working with the source code, for example developers, or users of \DFLOWFM's API.

The recommended way of using this guide is either via the index on page \pageref{ch:index}, or via the search function of your viewer.
